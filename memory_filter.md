# Memory Filter

## Store Only
- Stable style preferences
- Permanent rules
- Long-term goals
- Important decisions

## Never Store
- Temporary requests
- Emotional expressions
- One-time tasks
- Context-specific details

## Maintenance
- Review memory weekly
- Remove outdated information
- Keep only what truly matters

## Memory Types
### Permanent Memory
- User preferences and style guides
- Project-specific rules and constraints
- Long-term goals and objectives
- Important architectural decisions

### Temporary Memory
- Single task requests
- Context-specific details
- Emotional expressions
- Temporary working state

## Memory Cleanup Protocol
1. Regular review of memory files
2. Remove outdated or irrelevant information
3. Consolidate related memories
4. Preserve only what provides long-term value
